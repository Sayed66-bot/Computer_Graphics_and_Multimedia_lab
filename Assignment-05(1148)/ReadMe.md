# OpenGL Animated Rectangle

A simple OpenGL project using GLFW, GLAD, and GLM that renders an animated rectangle with:

- Smooth color animation (Red ↔ White)
- Rotation animation
- Scaling animation
- Translation animation

---

## Features

- OpenGL 3.3 Core Profile
- Custom Vertex & Fragment Shaders
- Rectangle created using 2 triangles
- Real-time transformations using GLM
- Dynamic color changes using sine wave animation

---

## Technologies Used

- OpenGL
- GLFW
- GLAD
- GLM
- stb_image

---

## Animation Effects

### 1. Color Animation
The rectangle smoothly changes between:

- Red
- White

Using:
cpp
float greenValue = sin(timeValue) / 2.0 + 0.5;
